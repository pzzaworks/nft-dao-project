# NFT DAO Project

A blockchain-based project that combines NFTs (ERC721 and ERC1155 tokens) with DAO (Decentralized Autonomous Organization) voting functionality.

## Features

- ERC721 NFT Contract Implementation
- ERC1155 NFT Contract Implementation
- DAO Voting Contract
- NFT Generator Tool

## Project Structure

- `/contracts`: Smart contract implementations
  - `ERC721Contract.sol`: Standard ERC721 NFT implementation
  - `ERC1155Contract.sol`: Standard ERC1155 NFT implementation
  - `VotingContract.sol`: DAO voting mechanism
- `/nft-generator`: Tools for NFT generation
- `/scripts`: Deployment and utility scripts

## Technologies

- Hardhat
- OpenZeppelin
- Solidity
- Ethereum

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

1. Clone the repository
   ```
   git clone https://github.com/yourusername/nft-dao-project.git
   cd nft-dao-project
   ```

2. Install dependencies
   ```
   npm install
   ```

3. Compile contracts
   ```
   npx hardhat compile
   ```
